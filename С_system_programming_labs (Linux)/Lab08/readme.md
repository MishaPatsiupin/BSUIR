# Лабораторная работа №8: Сокеты. Взаимодействие процессов.

## Описание
Эта лабораторная работа демонстрирует взаимодействие процессов с использованием сокетов. Программа состоит из сервера и клиента, которые могут общаться друг с другом через команды.

## Установка
Сначала соберите программу с помощью makefile:
```bash
make
```

## Запуск
Для запуска сервера и клиента откройте два разных окна терминала.

Запуск сервера:
```bash
./server
```

Запуск клиента:
```bash
./client
```

При запросе IP-адреса сервера введите `localhost`, если вы работаете на одной машине. Если вы работаете в локальной сети, узнайте IP-адрес сервера с помощью команды `ip a` до запуска сервера.

## Команды
Список доступных команд:

- `pwd`: Выводит рабочий каталог сервера.
- `echo`: Проверяет сообщение с клиентом.
- `info`: Выводит информацию о сервере.
- `cd`: Изменяет рабочий каталог сервера.
- `del`: Удаляет файл на сервере.
- `mkdir`: Создает новый каталог на сервере.
- `rmdir`: Удаляет каталог на сервере.
- `ls`: Выводит список файлов в текущем каталоге сервера.

Вы всегда можете получить этот список команд из сервера с помощью команды `help`.
